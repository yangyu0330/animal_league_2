# MVP 범위 확정

## 목표
- 6주 MVP에 필요한 최소 기능만 확정한다.
- 2주차 구현 시작 시 의사결정 질문이 나오지 않게 한다.

- [1주차 실행 계획서](../../../1주차%20실행%20계획서.md)

## 이번 주 고정 범위 (In Scope)
- 소셜 로그인: Google OAuth 1종
- 학교 선택 후 학과 검색
- 학과 미존재 시 신규 등록
- 학과 상세 진입 및 클릭
- 전국 랭킹/학교별 랭킹 조회
- 내 최근 클릭 기록 조회
- 공유 링크 진입

## 제외 범위 (Out of Scope)
- 교수/강의/과목 단위 기능
- 댓글/게시판/리뷰
- 즐겨찾기
- 관리자 전용 웹 CMS
- 마스코트 AI 생성
- 밀도 지표 기반 랭킹

## 핵심 규칙
- 랭킹 기준은 총 클릭 수 단일 지표로 고정
- 학생 스택: `floor(total_clicks / 1000)`
- 압박 단계: 0~4 고정 구간
- 클릭은 자유 허용, 오토클릭 패턴만 랭킹 합산 제외
